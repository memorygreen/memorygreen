### 6. 핵심 기능
이 서비스의 핵심 기능은 학교 인증과 청소년 연령제한, 욕설/비속어 필터링 기능입니다. 
사용자는 학생증 인증을 통해 학교 인증 후 서비스를 이용할 수 있으며, 청소년 연령을 제한할 수 있습니다.
또한 채팅 스타일의 댓글기능을 구현하여 사용자가 채팅을 하는 듯 몰입하여 댓글을 작성할 수 있습니다.
